# Ouija Pad - Frontend

This directory contains the frontend source files for the Ouija Pad application.

## Files

- `index.html` - Main HTML structure
- `style.css` - Gothic styling and animations
- `script.js` - Planchette logic and input handling
- `ouija.webp` - Ouija board image asset

## Development

The frontend is served by Tauri's development server. No separate build step is required.

## Fonts

The application uses Google Fonts:
- Creepster (titles)
- Nosifer (accents)
- Special Elite (text editor)
- Inter (UI elements)
